Semantic Html

Semantic html elements are specific html tags that describe the type of content they contain eg h1,h2,h3,h4,h5,h6 p,header,main,article,section,ol ,ul,input,form label,

Non semantic html

non-semantics html elements refers to html tags that do not inherently convey any specific meaning or purpose they include div,span,b,i,u,br,center,strike,del
